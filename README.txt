Программа DzmG1.exe предназначена для расчета параметров делительной защиты синхронного генератора, на основе математического моделирования переходного процесса в синхронном генераторе при нарушении связи с энергосстемой и симметричном набросе (сбросе) нагрузки на основе численного решения дифференциальных уравнений методом Рунге-Кутта четвертого порядка.

Для реализации математической модели использована среда программирования Borland C++ Builder 6.0.

В файле Calculation.cpp содержится основной блок расчетов.

Initial.dat – файл импорта исходных данных.

Report.rez – файл-отчет, используемый для вывода мгновенных значений переменных, используемых в математической модели, а также результатов расчета параметров делительной защиты.

В математической медели генератора использованы научные наработки математического моделирования в среде ФОРТРАН, разработанные  д.т.н. Новашем Владимиром Ивановичем
